## 💡 KEY DECISIONS & INSIGHTS

### **Typography Impact:**
- Line-height 1.7 (vs 1.5) = massively more readable
- Letter-spacing 0.01em = subtle but critical
- Consistent spacing creates professional hierarchy
- Mobile line-height 1.2 prevents overlap

### **SVG Strategy:**
- All graphics are inline SVG (no external files)
- Hand-drawn = imperfect lines, slight rotations
- Low opacity for subtlety (discovered details)
- Minimal file size, maximum character

### **Animation Philosophy:**
- Subtle, not flashy
- Smooth easing (ease-out)
- Direction-aware (feels natural)
- 250ms sweet spot (fast but noticeable)

### **KITH-Level Details:**
- Paw prints = unexpected + personal (Noah's dog)
- Different sizes/rotations = feels authentic
- Margin notes = like Harrison wrote them himself
- Coffee ring = realism (overlapping circles)
